# TaniAI Nexus — Platform AI Pertanian Indonesia

Platform AI lengkap untuk petani Indonesia: diagnosis tanaman, asisten AI, cuaca, komunitas, dan notifikasi WhatsApp.

## Tech Stack

- **Backend**: Node.js + Express + Groq AI + Baileys (WhatsApp)
- **Frontend**: React 19 + Vite 7 + TypeScript
- **Database**: Supabase (PostgreSQL)
- **AI**: Groq (Llama 3.3-70B + Llama 3.2-90B Vision)
- **Weather**: OpenWeather API

## Setup Local Development

### 1. Prasyarat

- Node.js 18+
- Akun Supabase
- Akun Groq Console (untuk API key)
- Akun OpenWeather (opsional)

### 2. Install dependencies

```bash
# Backend
npm install

# Frontend
cd frontend && npm install
```
